Description du poste

TechSolutions recherche un développeur Front-End React passionné pour rejoindre notre équipe dynamique. Vous travaillerez sur des projets innovants dans un environnement stimulant.

Vous serez chargé de développer des interfaces utilisateurs réactives et intuitives pour nos clients, en collaborant étroitement avec nos designers et développeurs back-end.

Missions

  • Développer des interfaces utilisateurs modernes avec React
  • Collaborer avec les designers UX/UI pour implémenter les maquettes
  • Assurer la qualité du code et sa maintenabilité
  • Participer aux revues de code et aux réunions d'équipe
  • Optimiser les performances des applications

Profil recherché

  • Formation en développement web ou informatique
  • Connaissance approfondie de JavaScript, HTML et CSS
  • Expérience avec React et Redux (ou équivalent)
  • Sensibilité à l'UX/UI et au responsive design
  • Esprit d'équipe et bonnes capacités de communication

Compétences requises

React JavaScript HTML5 CSS3 Redux Git

À propos de l'entreprise

TechSolutions est une entreprise spécialisée dans le développement de solutions web innovantes pour les PME et grandes entreprises.
